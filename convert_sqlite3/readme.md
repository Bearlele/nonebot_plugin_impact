# 这是一个把这个小游戏json数据转换成sqlite3的脚本


## 使用方法
- 1. 把这个脚本放到和json文件同一个目录下(这个插件数据存储在bot/data/impact下, 路径参考)
```tree
├── bot
│   ├── data
│   │   └── impact
│   │       ├── userdata.json
│   │       ├── groupdata.json
│   │       ├── ejaculation_data.json
│   │       ├── convert_sqlite3.py
```
- 2. 在这个目录下打开命令行
- 3. 输入`python convert_sqlite3.py`即可
